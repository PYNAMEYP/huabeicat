# huabeicat

华北的一只猫的工具集合

## 工具列表

### 1. 社保计算器 (calc.html)
用于计算社保缴纳金额的工具。

**功能：**
- 根据工资计算养老、医疗、失业、工伤、生育保险
- 计算公积金
- 显示个人和公司缴纳金额

### 2. OBJ网格简化工具 (obj-simplifier.html)
用于简化3D OBJ模型文件的工具，可以减少网格复杂度。

**功能：**
- 支持拖拽上传或选择OBJ文件
- 解析OBJ文件结构（顶点、面、纹理坐标、法线）
- 可调节简化级别（10%-100%）
- 选项：
  - 移除纹理坐标
  - 移除法线
  - 合并重复顶点
- 实时显示统计信息
- 下载简化后的OBJ文件

**使用方法：**
1. 打开 `obj-simplifier.html`
2. 上传或拖拽OBJ文件
3. 调整简化选项
4. 点击"简化网格"
5. 下载简化后的文件
